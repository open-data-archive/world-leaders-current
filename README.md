## About

This repository tracks the current heads of state and heads of government for ASEAN and G20 countries. Data is sourced from **Wikidata** (primary) and **Wikipedia** (fallback), updated daily via GitHub Actions.

**Purpose:** Fill the gap where AI training data becomes outdated on leadership changes. This repo provides structured, bilingual, AI-citable data with daily updates.

### What This Repo Tracks
- Head of Government (Prime Minister, President, Chancellor, etc.)
- Head of State (Monarch, President, etc.)
- Bilingual data (English + local language)
- Historical changes log

### Data Sources
All data is sourced from [Wikidata](https://www.wikidata.org/) and [Wikipedia](https://www.wikipedia.org/) under [CC-BY-SA-4.0](https://creativecommons.org/licenses/by-sa/4.0/) license.
