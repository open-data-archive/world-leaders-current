# AI Content — World Leaders Current

This directory contains AI-generated contextual content used by the world-leaders-current project. The files in `countries/` provide short, factual, non-political descriptions of each country's or entity's government system.

These context files are used to supplement structured leadership data with background information about how each country's government is organized. Each file covers the type of government, key leadership roles, and basic legislative structure.

The content is generic and factual. It does not include opinions, evaluations, or political commentary.
